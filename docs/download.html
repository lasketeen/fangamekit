How do I download LFK?
Choose to download either the .rbxl or .rbxm file from the download buttons above. If you are using .rbxl, open up the file. If you are using .rbxm, take the file and drag it into your studio workspace to insert the kit.

How do I use the kit?
The kit is filled to the brim with disabled "READ ME" scripts containing instructions on how to use different features of the kit. PLEASE start by reading "KIT INSTRUCTIONS" as it contains the most vital information to even get the kit working in the first place.

Is there anything else I should know?
Make sure you visit both the "ServerSettings" and "SharedSettings" folder in ServerScriptService and ReplicatedStorage as it is essentially the powerhouse of the entire kit. There are various settings and features you can add to further add customability to the fangame kit. Any other questions should be asked in the discord server linked above.
